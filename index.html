• I polished my previous game in several ways. First, I had to make the game more organized,and re-playable, so I added game states. Now there is a preload, a main menu, and a reset button to bring you back to the menu. This makes for a more presentable, and fluid game. Also, I implemented the option to play as two different characters (Richio and Dinio). These players have different physics similar to the old Mario and Luigi dynamic in Super Mario Bros, which this game is obviously influenced by. Finally, I made the game more challenging by adding more enemies -- squirrels that jump up and down at different heights. The game will undoubtedly take players several times to beat. (I also updated with some neat new sound effects) ENJOY!
